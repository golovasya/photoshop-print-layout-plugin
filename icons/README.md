# Icons

В этой папке должны находиться иконки плагина:

- `icon-light.png` - 23x23px (светлая тема)
- `icon-light@2x.png` - 46x46px (светлая тема retina)
- `icon-dark.png` - 23x23px (тёмная тема)
- `icon-dark@2x.png` - 46x46px (тёмная тема retina)

## Создание иконок

1. Создайте изображение 23x23px в Photoshop/Illustrator
2. Экспортируйте в PNG с прозрачным фоном
3. Для retina версий увеличьте до 46x46px
4. Для светлой темы используйте тёмные цвета
5. Для тёмной темы используйте светлые цвета

## Временное решение

Если у вас нет иконок, можно создать простые заглушки 23x23px с текстом "PL" (от Print Layout).
